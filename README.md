# 🎬 ReelPulse — Social Media Reels Analytics Dashboard

## 📌 Project Overview

ReelPulse is a data analytics project designed to analyze user behavior and the social impact of short-form video platforms such as Instagram Reels, TikTok, and YouTube Shorts.

The project focuses on understanding:

* User engagement with reels content
* Psychological and social effects of short-form videos
* Purchasing behavior influenced by reels
* Usage patterns across different demographics
* Platform preferences and user segments

The project combines:

* Data Analysis
* Data Visualization
* Interactive Dashboards
* Business Intelligence
* User Behavior Analytics

# 📊 Dashboard Features

## 🏠 Overview Dashboard

Provides a quick summary of the platform statistics including:

* Total users
* Gender distribution
* Most used platforms
* Average daily usage
* User segments

## 🧠 Social & Psychological Impact Dashboard

Analyzes the emotional and psychological effects of reels usage:

* Productivity impact
* Addiction indicators
* Difficulty closing apps
* Emotional influence
* Social comparison behavior

## 🛍️ Purchase Behavior Dashboard

Explores how reels affect purchasing decisions:

* Purchase conversion rate
* Influencer impact
* Product categories
* Impulse buying behavior
* Consumer trends

## 🎯 Interactive Filters

Users can filter dashboards by:

* Gender
* Age group
* User segment
* Platform
* Occupation

---

# 📈 Key Insights

Some insights generated from the project include:

* Short-form videos significantly affect purchasing behavior.
* Younger users spend more time consuming reels content.
* High screen time correlates with productivity reduction.
* Social media platforms influence emotional well-being differently.
* Entertainment content dominates user preferences.
